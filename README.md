# 🚗 AI-Powered Vehicle Comparison & Recommendation System

Developed an AI-powered web application that enables users to compare cars and bikes using intelligent AI-generated insights. The application provides detailed vehicle analysis based on engine specifications, mileage, top speed, safety features, pricing, pros & cons, and overall recommendations.

Built using Python, Streamlit, and Groq API, the system integrates the Llama 3.1 large language model to generate structured and informative vehicle comparisons in real time. The application also includes dynamic vehicle image rendering, responsive layouts, and interactive filtering options for enhanced user experience.

## Key Features

* AI-powered vehicle comparison
* Real-time AI-generated recommendations
* Vehicle image integration
* Responsive and user-friendly interface
* Sidebar filters for fuel type, budget, and vehicle category
* Structured comparison reports with modern UI

## Technologies Used

* Python
* Streamlit
* Groq API
* Llama 3.1
* Pillow
* Requests
* Python-dotenv

## Project Objective

The primary objective of this project is to demonstrate practical implementation of AI integration, prompt engineering, API handling, and modern frontend development using Streamlit.
